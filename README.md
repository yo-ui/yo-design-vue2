# Youi-Vue2.0 前端UI框架 1.0

# 目录结构
```
|---- assets  资源目录 （样式、字体）
|----|---- less   css 预编译目录
|---- build  编译目录
|----|---- template.js  模版文件生成   node template 组件名  模块名
|----|---- template  模板目录
|---- examples  框架说明和demo目录
|---- node_modules 编译使用的包目录   （忽略文件）
|---- src  组件所在目录
|----|---- index.js 全组件输出
|----|---- components 组件目录
|---- .eslintignore  js 检测忽略文件配置
|---- .eslintrc.js   js检测相关配置
|---- .prettierrc.js  文件格式化配置
|---- package.json   项目管理配置
|---- README.MD    项目说明文件
```
